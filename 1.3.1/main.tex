\section{Аннотация:} 
\textbf{Цель работы:} экспериментально получить зависимость между напряжением и деформацией (закон Гука) для двух простейших напряженных состояний упругих тел: одноосного растяжения и чистого изгиба; по результатам измерений вычислить модуль Юнга. \\
\textbf{В работе используется:} В первой части - прибор Лерматова, проволока из исследуемого материала, зрительная труба со шкалой, набор грузов, микрометр, рулетка; во второй части - стойка для изгибания балки, индикатор для измерения величины прогиба, набор исследуемых стержней, грузы, линейка, штангенциркуль.

\section{Теоретичиские сведения}

\subsection{Определение модуля Юнга по измерениям растяжения проволоки}


Последовательность действий при вычислении модуля Юнга
\begin{enumerate}
\item Вычислим площадь поперечного сечения проволоки.
\[S =\dfrac{ \pi (\overline{d})^2}{4}\]
\[\varepsilon_S = S\sqrt{2\left( \dfrac{\sigma_d}{d}\right) ^2}\]

\item Найдем длину проволоки при помощи линейки. ( $\sigma_{L} = 0,05 см$ )
\item Направим зрительную трубу на зеркальце, а затем измерим изменение длины проволоки
\[\Delta L =\dfrac{\Delta l}{2h}\]
\[ \varepsilon_{\Delta L} = \Delta L\sqrt{\left( \dfrac{\sigma_{\Delta l}}{\Delta l}\right)^2 + \left(\dfrac{\sigma_d}{d}\right)^2+\left(\dfrac{\sigma_h}{h}\right)^2} \]
где $r$ - длина рычага, $\Delta l$ удлинение проволоки, $h$ - расстояние от шкалы до основания зеркальца.
\item Найдем предельный вес, который можно подвесить на проволоке. $\sigma_{\text{предельное}} = 900 \text{ Н}/\text{мм}^2$. $P_{\text{предельное}} = 0,3 \sigma_{\text{предел}} S \approx 44,8 H$.
\item Начнем навешивать по одному грузу, при этом оставляя суммарную нагрузку на балку постоянной. Снимем зависимость удлинения от массы навешиваемых грузиков и занесем данные в таблицу.
\item Построим график зависимости удлинения проволоки от нагрузки, по формуле наименьших квадратов построим прямую и по ней найдем жесткость проволоки..
\item По найденной жёсткости проволоки найдем модуль Юнга по формуле
\[E = \dfrac{k*l_0}{S}\]
\[\varepsilon_E = \sqrt{\left( \dfrac{\sigma_{k}}{k} \right)^2 + \left( \dfrac{\sigma_{S}}{S} \right)^2 + \left( \dfrac{\sigma_{l_0}}{l_0} \right)^2 }\]
\end{enumerate}


\newpage
\subsection{Определение модуля Юнга по измерениям изгиба палки}

Погрешности измерени длины линейкой $= 0,05$ см.\\
Последовательность действий при измерении модуля Юнга при изгибе палки: \\


\begin{enumerate}
\item Измерим L.
\item Измерим толщину и ширину стержней, занесем данные в таблицу. 
\item Кладем одну из балок ровно посередине упоров, чтобы сила прикладывалась ровно в центр. 
\item Сместим балку влево на 3 см, чтобы убедится в том, что результаты измерений будут разные при неправильном расположении.
\item Перевернем балку и проделаем то же самое. 
\item Эти операции проделываем для латунной, деревянной и стальной балок, занося все результаты измерений в таблицу. 
\item Построим графики для каждой балки, опираясь на данные из таблиц.
\item По наклону графиков определяем средние значения модулей Юнга по формуле (табл.5)
\[E = \dfrac{Pl^3}{4ab^3d_{max}}\]
\[\varepsilon_E = \sqrt{3 \left( \dfrac{\sigma_{l}}{l} \right)^2 + \left( \dfrac{\sigma_{P/d_{max}}}{P/d_{max}} \right)^2 + \left( \dfrac{\sigma_{a}}{a} \right)^2 + 3 \left( \dfrac{\sigma_{b}}{b} \right)^2}\]
\end{enumerate}

\section{Методика измерений}
В каждом опыте грузы поочередно навешиваются на соответствующий подвес. Особенно важно в первом опыте не класть грузы на стол, а вешать их на нижележащий подвес, чтобы суммарная сила действующая на балку сохранялась. Иначе она будет прогибаться и измерения будут неточными.
Все измерения заносятся в таблицу Exel, где далее подлежат обработке.
